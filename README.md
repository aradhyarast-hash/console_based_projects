**# Expense Tracker (C++)**



**A console-based expense tracker built in C++.**



**## Features**

**- Add expenses**

**- Store expenses in file**

**- Calculate total spending**



**## Project Structure**

**src/  -> source code**

**data/ -> stored data**



**## Compile**



**g++ src/expense\_tracker.cpp**



**## Run**



**./a.out**
